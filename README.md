# Vibe Coding 101

This is the repository for the Vibe Coding 101 educational platform.

## Vision

Vibe Coding 101 will teach people how to turn ideas into useful products by
working efficiently with AI. The platform will combine:

- **The book:** A practical guide to efficient vibe coding.
- **The website:** Structured lessons, examples, and progress tracking.
- **The community:** A Discord server where students can learn together and
  share what they build.
- **The marketplace:** A place where students can sell their creations and
  learn how to take a project from an idea to something valuable.
- **The evidence:** Transparent build records showing what the student
  directed, what the agent completed, and when human intervention was needed.

## Build in public

This project is being created primarily through relaxed, voice-driven direction
from a phone. See the [build log](BUILD_LOG.md) for the decisions, actions, and
manual computer interventions recorded along the way.
